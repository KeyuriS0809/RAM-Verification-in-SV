# RAM Verification using SystemVerilog

This project contains a SystemVerilog verification environment for RAM.

## Features
- Generator
- Driver
- Monitor
- Scoreboard
- Assertions
- Randomization
- Multiple testcases

## Testcases
- sanity_test
- single_write_read
- simultaneous_rw
- walk_0
- walk_1
- full_ram_check

## Tools Used
- Riviera-PRO
- SystemVerilog
